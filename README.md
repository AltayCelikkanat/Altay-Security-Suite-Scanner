# Altay-Security-Suite-Scanner
Altay Security Suite; siber güvenlik odaklı, profesyonel bir Python araç setidir. AES-256 şifreleme sunan Altay-Crypto ve çoklu iş parçacıklı port taraması yapan Altay-Scanner modüllerini içerir. Kendi ağınızı denetlemek ve verilerinizi korumak için tasarlanmış; şeffaf, güvenli ve performans odaklı, mimarisiyle modern bir siber güvenlik çözümüdür.
